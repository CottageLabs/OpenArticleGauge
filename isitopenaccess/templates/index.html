OpenArticleGauge

The OpenArticleGauge is a service for determining the licence status of journal publications, providing an end-user interface for looking up lists of identifiers, and an API for inclusion into third-party software systems. Cut and paste a DOI or Pubmed ID (or a list) into the box to search for licensing information.
